**Description**
The program sort a list of numbers whith an algorithm in only two lists. In this case I choose Radix algorithm.

**Instrucctions**
If you want to use this program, you must compile and with the executable you can give to the program all the numbers that you want by arguments. You have to know:
-The arguments have to be numbers.
-If you put something that is not a number or put a number two or more times you will see an error on your screen.
-You can give the numbers one by one or all that you want in a string. Example: 1 4 6 23 5 2 or "15 35 65 23 17 62".
-If you put a number bigger or smaller than the limits of an integer, you will see an error.
-If you give a sorted list (1 2 3 4 5) you will not see nothing because the arguments are sorted.
-After you give to the program the list of number you will see some movements. For a better understunding, we'll going to name A to the first list and B to the secondary list.  Here you have all the posibilites:
	·pa: The first number of the list B go to the first position on the list A.
	·pb: The first number of the list A go to the first position on the list B.
	·sa: Swap the first two numbers on the list A.
	·sb: Swap the first two numbers on the list B.
	·ss: Swap the first two numbers on the list A and B at the same time.
	·ra: The first number of the list A go to the end of the list and this number now will be the la		st one.
	·rb: The first number of the list B go to the end of the list and this number now will be the la		st one.
	·rr: The first number of the list A and B o to the end of their lists and this number now will b		e the last one.
	·rra: The last number of the list A go to the beggining of the list and this number now will be 		the first one.
	·rrb: The last number of the list A go to the beggining of the list and this number now will be 		the first one.
	·rrr: The last number of the list A and B go to the beggining of their lists and this number now 		will be the first one.
-After you see all the movements you must have a sort list.
